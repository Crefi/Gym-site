My projects in C

The application allows the user to register, log in and log out. Every user has a list of channels to which it subscribes. It can search for a channel, subscribe or unsubscribe from one and can assign a rating from 1 to 5 to that channel according to their own preference. The application will allow the display of a top 25 channels depending on the rating received (regardless of whether the user is logged in or not).
